# WebP Desktop Converter (Python CLI)

這是一個專為提升前端開發效率設計的桌面自動化工具。

- **擺脫限制**：無需依賴線上轉換器，免除上傳等待與轉換次數限制。
- **極簡操作**：將圖片拖拽至程式圖示即可完成轉換與優化，流程直覺。
- **隱私安全**：所有影像處理皆在本地端完成，無需上傳至雲端伺服器。

## 💡 開發初衷

在前端開發過程中，圖片優化（LCP 優化）是關鍵的一環。
我希望建立一個「無感」的自動化流程，讓原本繁瑣的「開啟線上轉換器 -> 上傳 -> 下載 -> 改名」過程，簡化成一個簡單的拖拽動作。

這也是我探索 Python 自動化腳本與影像處理的一個練習作品。

## ✨ 核心功能

- **自動化處理**：支援 `JPG`, `PNG`, `BMP`, `TIFF` 格式，拖拽至 `.exe` 即可執行。
- **智能路徑管理**：自動偵測桌面路徑，並整理產出至 `WebP_Converted` 資料夾。
- **智能體積優化**：
  - **> 1MB**：自動執行深度壓縮 (Quality: 70)，確保效能。
  - **< 1MB**：維持高品質轉換 (Quality: 90)，保留視覺細節。
- **極致壓縮比**：採用 `method=6` 演算法，追求同畫質下的最小體積。
- **操作反饋**：轉換完成後自動開啟輸出資料夾，流程不中斷。

## 🛠️ 技術棧

- **Language**: Python 3.12+
- **Library**: `Pillow` (PIL) - 專業影像編碼與壓縮處理。
- **Tool**: `PyInstaller` - 環境封裝與執行檔打包。

## 🚀 快速開始

#### 對於一般使用者 (Windows)

1.  前往本專案的 [Releases](https://github.com/153ys/webp_converter/releases/tag/v1.0.0) 頁面下載最新版本的 `webp_converter.exe`。
2.  將檔案放置於桌面。
3.  選取欲轉換的圖片，直接拖拽到 `webp_converter.exe` 圖示上。
4.  桌面將自動彈出包含新 WebP 檔案的資料夾。

#### 對於開發者 (本地運行)

如果您想自行修改邏輯或運行腳本：

```bash
# 安裝依賴
pip install Pillow

# 運行腳本
python webp_converter.py [圖片路徑]
```
